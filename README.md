# Citizen Complaint Resolution

A municipal service platform built on Stacks blockchain for managing citizen complaints with comprehensive issue tracking and resolution workflow.

## Key Features
- **Issue categorization**: Infrastructure, sanitation, public safety, utilities, and other complaint types
- **Department routing**: Automatic assignment and staff permission management
- **Progress tracking**: Real-time status updates from submission to resolution
- **Satisfaction measurement**: Post-resolution citizen rating system (1-5 stars)
